How to Host a Webpage on the Pigrow

To host a webpage using the Pigrow, you need to point the host_webpage.py script to the appropriate folder containing your HTML, CSS, and JavaScript files. By default, Flask serves HTML files from a folder named 'templates' and static files, such as CSS and JavaScript, from a folder named 'static'. To display your own pages, create an 'index.html' file in the 'templates' folder and place your CSS and JavaScript files in the 'static' folder.

It is possible to use CSS and JavaScript in your webpages. Simply include the appropriate link and script tags in your HTML file. You can use the {{ url_for('static', filename='') }} template function to refer to the static files in your HTML.

Important things to be aware of:

  • Ensure that your HTML files are placed in the 'templates' folder and your CSS, JavaScript, and other static files are placed in the 'static' folder.
  • Include the correct paths to your CSS and JavaScript files using the {{ url_for() }} template function.
  • For any changes to take effect, you may need to restart the Flask server by stopping and starting the host_webpage.py script.
